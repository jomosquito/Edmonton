Select from the following forms to submit a request to the University of Houston. Make sure to complete all required fields and upload any necessary documentation.

Medical/Administrative Term Withdrawal

Use this form to request a complete withdrawal from all courses for medical or administrative reasons. This request requires supporting documentation and approval from multiple departments.

For medical withdrawals, you'll need documentation from a healthcare provider. For administrative withdrawals, appropriate documentation will be required based on the nature of your request.
